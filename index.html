Controls:

  • ⬅️➡️⬆️⬇️: Change cube selection on a layer
  • Shift + ⬆️⬇️: Move to a new layer
  • Numbers: Change the input of the selected cube

Rules:

  • Like Sudoku, fill the entire grid with numbers such that each row, column, and layer contains no repeated numbers.
  • Like KenKen, each cage's operands (highlighted in red) must equal the displayed result. E.g. 5+ means that all 3 cubes must add to 5. 1- means that the 2 cubes must subtract to 1.
